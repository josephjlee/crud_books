# crud_books
Create a sample CRUD on a list of books( Codeigniter Framework, Jquery, Bootstrap, Datatable, Datepicker, Mysql)

# Introduction step for application operation
1. Move this project to your local (example: xampp/htdocs)
2. Import sql (mplus_test.sql) in your database
3. setting config database.php in path this project /application/config/database.php <br>
	'hostname' => 'localhost',<br>
	'username' => 'root',<br>
	'password' => '',<br>
	'database' => 'mplus_test',<br>
	example default. if you make db with name is mplus_test
4. Run project in your browser. 
5. if you have been run, it will show like this :
