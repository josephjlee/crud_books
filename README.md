# crud_books
Create a sample CRUD on a list of books( Codeigniter Framework, Jquery, Bootstrap, Datatable, Datepicker, Mysql)

# Introduction step for application opertaion
1. Move this project to your local (example: xampp/htdocs)
2. Import sql (mplus_test.sql) in your database
3. setting config database.php in path this project /application/config/database.php
	'hostname' => 'localhost',
	'username' => 'root',
	'password' => '',
	'database' => 'mplus_test',
	example default. if you make db with name is mplus_test.sql
4. Run project in your browser.
